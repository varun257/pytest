Unit test - As a service

Pytest is designed with the intention to have a pytest microservice which can be used by any python 2.7 based project to execute test-cases based on pytest and generate the reports in html format.

This will facilitate faster dev while ensuring code quality both locally or on sdev, sdev preferred

SCM user that has public token (https) read access to the project on which it intends to run the tests
